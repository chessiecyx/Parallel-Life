🌲 Parallel Tree

A tree that takes 85 years to grow.

It started on January 23, 2005, 10:00 AM.

It will finish when a life statistically does.


Time does not feel like passing.

Growth is almost invisible.

Like a tree — always changing, never moving.

This work makes that slowness measurable.


The tree is built from 90,000 pixels.

Each pixel belongs to a moment.

As time passes, pixels appear.

Nothing is stored.

Everything is calculated from now.


This tree is not a metaphor.

It is a second timeline.

If I disappear, it continues.

If you see it years later, you will know how far I might have lived.

Not as age — but as form.


Code is written for speed.

This code resists it.

A fast system, forced to produce something almost still.


Pixels are not natural.

They are discrete, mechanical, precise.

A life, reduced into units.

A tree, reconstructed through digital fragments.
